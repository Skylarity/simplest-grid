# Simplest Grid

## Installation

JavaScript

```javascript
import "simplest-grid";
```

SASS/SCSS

```scss
@import "~simplest-grid";
```

## Usage

Wrap your content in a `.container`. Use `.col-<number>` for a column (we've got 1-6). Use `.col-start-<number>` (again, 1-6) to offset your column inwards.
